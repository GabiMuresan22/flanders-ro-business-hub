Add Business Feature - Test Suite

Use this checklist to manually test the Add Business functionality.

Open Add Business Form →

Test 1: Form Validation - Empty Submission

Steps:
  1. Navigate to the Add Business page
  2. Click "Submit Business" without filling any fields
Expected Result:

Validation errors should appear for all required fields

  • Business Name error displayed
  • Owner Name error displayed
  • Email error displayed
  • Phone error displayed
  • Address error displayed
  • City error displayed
  • Postal Code error displayed
  • Category error displayed
  • Description error displayed
  • Terms checkbox error displayed

Test 2: Email Validation

Steps:
  1. Enter invalid email formats: "test", "test@", "@example.com"
  2. Observe validation messages
Expected Result:

Error message: "Please enter a valid email address."

  • Invalid email "test" rejected
  • Invalid email "test@" rejected
  • Valid email "test@example.com" accepted

Test 3: Website URL Validation

Steps:
  1. Enter invalid URL: "not-a-url"
  2. Enter valid URL: "https://example.com"
  3. Leave empty (optional field)
Expected Result:

Invalid URLs show error, valid URLs and empty field are accepted

  • Invalid URL rejected
  • Valid URL accepted
  • Empty field accepted (optional)

Test 4: Successful Form Submission

Steps:
  1. Fill all required fields with valid data:
    • Business Name: Test Romanian Restaurant
    • Owner Name: Ion Popescu
    • Email: test@romanianrestaurant.com
    • Phone: +32 470 123 456
    • Address: Markt 15
    • City: Bruges
    • Postal Code: 8000
    • Category: Restaurant & Food
    • Description: We serve authentic Romanian cuisine... (10+ chars)
    • Check terms checkbox
  2. Click "Submit Business"
Expected Result:
  • Button shows "Submitting..." while processing
  • Button is disabled during submission
  • Success toast notification appears
  • Form resets to empty values
  • Data saved to Supabase with status "pending"
  • Button text changes to "Submitting..."
  • Button disabled during submission
  • Success toast appears
  • Form resets after submission

Test 5: Character Length Validations

Steps:

Test minimum character requirements for each field:

  • Business Name: minimum 2 characters
  • Owner Name: minimum 2 characters
  • Phone: minimum 9 characters
  • Address: minimum 5 characters
  • City: minimum 2 characters
  • Postal Code: minimum 4 characters
  • Description: minimum 10 characters

Test 6: Category Selection

Steps:
  1. Open the category dropdown
  2. Verify all categories are listed
  3. Select a category
Expected Categories:

Restaurant & Food, Bakery, Grocery, Beauty & Wellness, Car Services, Construction, Transportation, Travel & Tourism, Retail, Professional Services, Other

  • All categories visible
  • Category selection works
  • Required validation enforced

Test 7: Terms and Conditions Checkbox

Steps:
  1. Fill all fields correctly
  2. Leave terms checkbox unchecked
  3. Try to submit
Expected Result:

Error message: "You must agree to the terms and conditions."

  • Unchecked terms prevents submission
  • Error message displayed
  • Checked terms allows submission

Test Summary

Complete the tests above and check all boxes to verify functionality.
