Custom Open Graph Image
======================

The site expects a custom OG image at:  /og-image.png  (this folder).

Requirements:
- Size: 1200 x 630 px (recommended for Facebook, LinkedIn, Twitter)
- Format: PNG or JPG
- Content: Branded image with Romanian Business Hub logo and tagline

Once created, save it as:  og-image.png  in the public folder (same folder as this file).

This image is used when the site is shared on social media (Facebook, Twitter, LinkedIn, etc.).
